# Other Tab

## 1. General
- **Auto Clear Message Box**: Enable or disable automatic clearing of the message box after a certain period.
- **Send Only Translated Message**: Choose whether to send only the translated message or both original and translated messages.
- **Auto Save Message Log**: Enable or disable automatic saving of message logs.
- **VRC Mic Mute Sync**: Enable or disable synchronization of mute status between VRChat and VRCT.
- **Send Message To VRChat**: Configure how messages are sent to VRChat (e.g., via chat or voice).

## 2. Sounds
- **Enable Notification Sound When Sending Chat**: Toggle notification sounds for sending chat messages.

## 3. Speaker2Chatbox
- **Send Receive Message To VRChat**: Toggle the Speaker2Chatbox feature on or off.

## 4. Message Format
- **Customize Message Format**: Define how messages are formatted when sent to VRChat.

## 5. Other
- **Show Romaji**: Enable or disable the display of Romaji for Japanese text.
- **Show Hiragana**: Enable or disable the display of Hiragana for Japanese text.
