# Translation Tab

- **CTranslate2 Model**: Select the local model for offline transcription/translation (if available).
- **Processing device for AI translation**: Choose between CPU or GPU for processing.
- **DeepL API Key**: Enter your API keys for services like DeepL or OpenAI if you are using their paid/pro features.
