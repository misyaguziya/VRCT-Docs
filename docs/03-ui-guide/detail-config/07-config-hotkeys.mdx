# HotKeys Tab
Customize various hotkey settings in the HotKeys tab of the VRCT configuration.

## Video

<TwitterEmbed url="https://twitter.com/Shiina_12siy/status/1882028408185712799" />

## General
![config-hotkey-overview.png](./img/config-hotkey-overview.png)

- **Toggle VRCT visibility**: Set a hotkey to show or hide the VRCT window.
- **Toggle Translation**: Set a hotkey to enable or disable translation functionality.
- **Toggle Voice2Chatbox**: Set a hotkey to enable or disable the Voice2Chatbox feature.
- **Toggle Speaker2Log**: Set a hotkey to enable or disable the Speaker2Log feature.
