# Appearance Customization

VRCT offers various customization options to personalize the look and feel of the application according to your preferences.

## Overview

You can customize:
- Window appearance and transparency
- Font sizes and styles
- Color themes
- UI layout
- Chat display options
- And more...

## Quick Customization

### Font Size Adjustment

The fastest way to adjust readability:

1. In the main window, click the **Font Size** button
2. A slider window will appear
3. Adjust the slider to change message log font size
4. Default size is 100

<!-- ![Font Size Slider](./img/font-size-slider.png) -->

:::tip[Tip]
Larger font sizes make messages easier to read but reduce the number of visible messages.
:::

## Appearance Settings

### Access Appearance Settings

1. Open the Settings window (click the gear icon)
2. Navigate to the **Appearance** section
3. Configure appearance options
4. Click **Save** to apply changes

For detailed appearance settings, see the [Appearance Settings Guide](/config-appearance).

## Customization Options

### Window Settings

**Window Transparency**:
- Adjust main window opacity
- Make VRCT semi-transparent
- Useful for seeing VRChat behind VRCT
- Range: 0% (invisible) to 100% (opaque)

**Window Style**:
- Bordered/Borderless
- Title bar visibility
- Window decorations
- Minimize/maximize/close buttons

**Always on Top**:
- Keep VRCT above other windows
- Quick toggle from main window
- Useful when switching between apps

## Tips for Better Appearance

### For Readability

1. **Sufficient Contrast**: Ensure text stands out from background
2. **Appropriate Font Size**: Not too small, not too large
3. **Clear Fonts**: Use readable font families
4. **Adequate Spacing**: Don't cram information

### For Performance

1. **Reduce Transparency**: Full opacity uses less GPU
2. **Simpler Themes**: Complex themes may impact performance
3. **Disable Animations**: If experiencing lag
4. **Optimize Shadows**: Reduce shadow effects

### For Streaming/Recording

1. **Clean UI**: Minimize distracting elements
2. **High Contrast**: Ensure visibility in recordings
3. **Appropriate Size**: Consider viewers' screen sizes
4. **Branding**: Customize colors to match your brand

## Troubleshooting

### Appearance Settings Not Applying

- Click **Save** button in settings
- Restart VRCT to apply some changes
- Check if settings file is read-only
- Reset to default and try again

### UI Elements Overlapping

- Reset to default layout
- Adjust window size
- Check UI scale setting
- Disable conflicting plugins

### Fonts Not Displaying Correctly

- Ensure font is installed on system
- Try default font
- Check font file permissions
- Update VRCT to latest version

### Performance Issues

- Reduce transparency
- Disable animations
- Use simpler theme
- Lower UI scale
- Reduce font effects (shadows, outlines)

## Default Settings

To reset appearance to default:

1. Open Settings window
2. Navigate to Appearance section
3. Click **Reset to Default**
4. Confirm the reset
5. Restart VRCT

## Sharing Themes
