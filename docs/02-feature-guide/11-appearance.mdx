# Appearance Customization

VRCT offers various customization options to personalize the look and feel of the application according to your preferences.

## Overview

You can customize:
- Language
- Font sizes and families
- IU elements
- Transparency

## Customization


### Language Selection
1. Open the Config Window (click the gear icon)
2. Navigate to the **Appearance** Tab
3. Select your preferred language from the radio buttons
    - English
    - Japanese
    - Korean
    - Chinese (Simplified)
    - Chinese (Traditional)
4. Click Save

### Font Size Adjustment

The fastest way to adjust readability:

1. In the main window, click the **Font Size** button. (Refer to the docs on [Main Window Font Size Slider](../main-window#message-log-font-size-slider) for button location)
2. A slider window will appear
3. Adjust the slider to change message log font size
4. Default size is 100

:::tip[Tip]
Larger font sizes make messages easier to read but reduce the number of visible messages.
:::

## Appearance Settings

### Access Appearance Settings

1. Open the Config Window (click the gear icon)
2. Navigate to the **Appearance** Tab
3. Configure appearance options

For detailed appearance settings, see the [Appearance Config Guide](../config-appearance).

## Customization Options

### Window Settings

**Window Transparency**:
- Adjust main window opacity
- Make VRCT semi-transparent
- Useful for seeing VRChat behind VRCT
- Range: 0% (invisible) to 100% (opaque)

### Font Customization

**UI/Message Font**:
- Choose font family
- UI font size 40%-200%
- Message log font size 40%-200%

### UI Elements

**Show/Hide Send Message Button**:
- Toggle visibility of the Send button in the input field

**Show/Hide Resend Button**:
- Toggle visibility of the Resend button in the Log field
