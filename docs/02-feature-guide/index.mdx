# Welcome to the Feature Guide

This feature guide will walk you through the various features and functionalities of VRCT.  
Follow the sections below to explore in-depth information:

## Core Features

### Translation Features
- **[Real-time Translation](/docs/feature-guide/translation)** - Translate messages in real-time using various translation engines
- **[Multiple Target Languages](/docs/feature-guide/multi-target-language)** - Support for translating to multiple languages simultaneously
- **[Translation Engine Selection](/docs/feature-guide/translation-engines)** - Choose from various translation engines (Google, DeepL, AI models, etc.)

### Transcription Features
- **[Voice to Text (Voice2Chatbox)](/docs/feature-guide/voice-to-text)** - Convert your voice to text using microphone
- **[Speaker to Text (Speaker2Log)](/docs/feature-guide/speaker-to-text)** - Transcribe system audio to text

### Communication Features
- **[Send Message](/docs/feature-guide/send-message)** - Send messages with translation via text or voice
- **[Receive Message](/docs/feature-guide/receive-message)** - Receive and translate messages from others
- **[OSC Integration](/docs/feature-guide/osc-integration)** - Integration with VRChat OSC for chatbox functionality

### VR Features
- **[Overlay Mode](/docs/feature-guide/overlay-mode)** - Display VRCT interface in VR as an overlay
- **[VRC Mute Sync](/docs/feature-guide/vrc-mute-sync)** - Synchronize mute status with VRChat

### Customization Features
- **[Language Presets](/docs/feature-guide/language-presets)** - Quick switching between preset language combinations
- **[Appearance Customization](/docs/feature-guide/appearance)** - Customize the look and feel of VRCT
