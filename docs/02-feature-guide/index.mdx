# Welcome to the Feature Guide

This feature guide will walk you through the various features and functionalities of VRCT.  
Follow the sections below to explore in-depth information:

## Core Features

### Translation Features
- **[Real-time Translation](./translation)** - Translate messages in real-time using various translation engines
- **[Multiple Target Languages](./multi-target-language)** - Support for translating to multiple languages simultaneously
- **[Translation Engine Selection](./translation-engines)** - Choose from various translation engines (Google, DeepL, AI models, etc.)

### Transcription Features
- **[Voice to Text (Voice2Chatbox)](./voice-to-text)** - Convert your voice to text using microphone
- **[Speaker to Text (Speaker2Log)](./speaker-to-text)** - Transcribe system audio to text

### Communication Features
- **[Send Message](./send-message)** - Send messages with translation via text or voice
- **[Receive Message](./receive-message)** - Receive and translate messages from others
- **[OSC Integration](./osc-integration)** - Integration with VRChat OSC for chatbox functionality

### VR Features
- **[Overlay Mode](./overlay-mode)** - Display VRCT interface in VR as an overlay
- **[VRC Mute Sync](./vrc-mute-sync)** - Synchronize mute status with VRChat

### Customization Features
- **[Language Presets](./language-presets)** - Quick switching between preset language combinations
- **[Appearance Customization](./appearance)** - Customize the look and feel of VRCT
