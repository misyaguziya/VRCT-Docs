# Configuration Window Overview

The Configuration Window allows you to customize various aspects of VRCT, from appearance to translation engines.   
Click the **gear icon (⚙️)** on the main window to open it.

![config-window-overview.png](./img/config-window-overview.png)

## 1. Device
Manage your audio input and output devices.  
refer to the [Device Tab](./02-config-device.mdx) page for more details.

## 2. Appearance
Customize the look and feel of the application.  
refer to the [Appearance Tab](./03-config-appearance.mdx) page for more details.

## 3. Translation
Configure the translation engines.  
refer to the [Translation Tab](./04-config-translation.mdx) page for more details.

## 4. Transcription
Configure the transcription engines.  
refer to the [Transcription Tab](./05-config-transcription.mdx) page for more details.

## 5. VR
Configure VR-related(mainly about Overlay) settings.  
refer to the [VR Tab](./06-config-vr.mdx) page for more details.

## 6. Other
Configure other miscellaneous settings.  
refer to the [Other Tab](./07-config-other.mdx) page for more details.

## 7. HotKeys
Configure keyboard shortcuts for various actions in VRCT.  
refer to the [HotKeys Tab](./08-config-hotkeys.mdx) page for more details.

## 8. Plugins
Configuration options for managing VRCT plugins.  
refer to the [Plugins Tab](./09-config-plugins.mdx) page for more details.

## 9. Advanced Settings
Advanced configuration options for VRCT.  
refer to the [Advanced Settings Tab](./10-config-advanced.mdx) page for more details.
