# VRCT Documentation

VRChat Chatbox Translation (VRCT) is a tool that enables real-time translation and transcription for VRChat. This repository contains the official documentation website for VRCT.

VRCT Documentation is built using [Docusaurus](https://docusaurus.io/), a modern static website generator. The documentation covers:

- **Installation & Setup**: Getting started with VRCT
- **Feature Guide**: Detailed guides for all VRCT features
- **UI Guide**: Overview of the main window and configuration options
- **FAQ**: Common questions and troubleshooting

## ðŸš€ Quick Start

### Prerequisites

- Node.js 20.0 or higher
- npm (comes with Node.js)

### Installation

```bash
git clone https://github.com/misyaguziya/VRCT-Docs.git
cd VRCT-Docs
npm install
```

### Local Development

```bash
npx docusaurus start
```

This command starts a local development server and opens a browser window at `http://localhost:3000`. Most changes are reflected live without having to restart the server.

### Build for Production

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static hosting service.

### Serve Built Site Locally

```bash
npm run serve
```

This command serves the built site locally for testing before deployment.
