# 翻訳付きメッセージの送信
このガイドでは、VRCTで翻訳付きメッセージを送信するプロセスを説明します。

## ステップ1: 翻訳機能を有効にする
   VRCTのメインウィンドウで、「翻訳」トグルをオンに切り替えてください。  
   ![step1](./img/send-message-step1.png)

    :::tip[tip]
    翻訳機能を有効にしない場合、メッセージは翻訳なしで送信されます。
    :::


## ステップ2: メッセージを送信する
メッセージを送信する方法は2つあります。

1. **チャット入力フィールドを使用する**  
   VRCTのメインウィンドウのチャット入力フィールドにメッセージを入力し、Enterキー（または送信ボタン）を押してメッセージを送信できます。  
   ![step2](./img/send-message-step2.png)

2. **マイクを使用する**  
   マイクを使用して、音声でメッセージを送信することもできます。VRCTはあなたの音声を認識し、テキストに変換して送信します。  
   ![step3](./img/send-message-step3.png)
    :::tip[tip]
    メッセージを送信する前に、VRCTの設定で使用するマイクとスピーカーが正しく選択されていることを確認してください。
    :::

3. **送信されたメッセージを確認する**  
   メッセージを送信した後、メインウィンドウのチャット履歴エリアで確認できます。翻訳されたメッセージは元のメッセージと一緒に表示されます。  
   ![step4](./img/send-message-step4.png)
