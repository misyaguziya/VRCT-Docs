# 翻訳付きメッセージの受信

このガイドでは、VRCTで翻訳付きメッセージを受信するプロセスを説明します。

## ステップ1: 翻訳機能を有効にする
    VRCTのメインウィンドウで、「翻訳」トグルをオンに切り替えてください。  
    ![step1](./img/receive-message-step1.png)

    :::tip[tip]
    翻訳機能を有効にしない場合、メッセージは翻訳なしで表示されます。
    :::

## ステップ2: メッセージを受信する

1. **スピーカーデバイスを使用する**  
    選択したスピーカーデバイスを通じて音声メッセージを受信できます。VRCTは音声メッセージを認識し、テキストに変換して、チャット履歴エリアに表示します。  
    ![step4](./img/receive-message-step2.png)

    :::tip[tip]
    メッセージを受信する前に、VRCTの設定で使用するマイクとスピーカーが正しく選択されていることを確認してください。
    :::

2. **受信したメッセージを確認する**  
    メッセージを受信した後、メインウィンドウのチャット履歴エリアで確認できます。翻訳されたメッセージは元のメッセージと一緒に表示されます。  
    ![step4](./img/receive-message-step3.png)

:::warning[warning]
VRCTがスピーカーを文字起こししている間にWindowsでスピーカーデバイスを変更すると、VRCTがフリーズする可能性があります。  
変更したい場合は、まずスピーカー文字起こし機能を停止してください。
:::
