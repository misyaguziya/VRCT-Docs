# 外観タブ
アプリケーションのルック・アンド・フィールをカスタマイズします。
![config-appearance-overview.png](./img/config-appearance-overview.png)

- **UI言語**：サポートされているUI言語間を切り替えます。
    - 英語、日本語、韓国語、中国語（簡体字、繁体字）
- **UIサイズ（%）**：アプリケーションインターフェースのサイズを調整します。
    - 40%～200%
    - デフォルト：100%
- **メッセージログフォントサイズ（%）**：メッセージログ領域のテキストサイズを変更して読みやすくします。
    - 40%-200% / デフォルト：100%
- **送信メッセージボタン**：送信メッセージボタンの表示を選択します。
    - 非表示：Enterキーのみを使用して送信します。
    - 表示：ボタンとEnterキーの両方を有効にします
    - 表示して、Enterキーを使用した送信を無効にします。
- **再送信ボタンを表示**：メッセージの再送信ボタンを有効または無効にします。
    - 送信されたメッセージログの上にカーソルを置くと、再送信ボタンが表示されます。
    - クリックして編集、長押しして再送信します。
- **フォントファミリー**：アプリケーションで使用されるフォントスタイルを選択します。
- **透明度**：ウィンドウの不透明度を調整して、邪魔を減らします。
    - 40%-100% / デフォルト：100%
