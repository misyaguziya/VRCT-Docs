# 機能ガイドへようこそ

この機能ガイドでは、VRCTのさまざまな機能と機能性を説明します。
以下のセクションに従って、詳細情報を確認してください：

## コア機能

### 翻訳機能
- **<Link to="/docs/feature-guide/translation">リアルタイム翻訳</Link>** - さまざまな翻訳エンジンを使用してメッセージをリアルタイムで翻訳します
- **<Link to="/docs/feature-guide/multi-target-language">複数の対象言語</Link>** - 複数の言語への同時翻訳をサポートしています
- **<Link to="/docs/feature-guide/translation-engines">翻訳エンジンの選択</Link>** - さまざまな翻訳エンジンから選択します（Google、DeepL、AIモデルなど）

### 文字起こし機能
- **<Link to="/docs/feature-guide/voice-to-text">音声からテキストへ（Voice2Chatbox）</Link>** - マイクを使用して音声をテキストに変換します
- **<Link to="/docs/feature-guide/speaker-to-text">スピーカーからテキストへ（Speaker2Log）</Link>** - システムオーディオをテキストに文字起こしします

### コミュニケーション機能
- **<Link to="/docs/feature-guide/send-message">メッセージ送信</Link>** - テキストまたは音声で翻訳付きメッセージを送信します
- **<Link to="/docs/feature-guide/receive-message">メッセージ受信</Link>** - 他者からのメッセージを受信して翻訳します
- **<Link to="/docs/feature-guide/osc-integration">OSC連携</Link>** - チャットボックス機能用のVRChat OSCとの統合

### VR機能
- **<Link to="/docs/feature-guide/overlay-mode">オーバーレイモード</Link>** - VR内にVRCTインターフェイスをオーバーレイとして表示します
- **<Link to="/docs/feature-guide/vrc-mute-sync">VRCミュート同期</Link>** - VRChatのミュート状態と同期します

### カスタマイズ機能
- **<Link to="/docs/feature-guide/language-presets">言語プリセット</Link>** - プリセット言語の組み合わせを素早く切り替えます
- **<Link to="/docs/feature-guide/appearance">外観のカスタマイズ</Link>** - VRCTの外観をカスタマイズします
