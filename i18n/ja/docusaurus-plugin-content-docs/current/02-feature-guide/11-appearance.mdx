# 外観のカスタマイズ

VRCTは、あなたの好みに応じてアプリケーションのルック・アンド・フィールをカスタマイズするためのさまざまなオプションを提供しています。

## 概要

カスタマイズできる項目：
- 言語
- フォントサイズとフォントファミリー
- UI要素
- 透明度

## カスタマイズ


### 言語選択
1. コンフィグウィンドウを開く（ギアアイコンをクリック）
2. **外観**タブに移動
3. ラジオボタンから希望の言語を選択
    - 英語
    - 日本語
    - 韓国語
    - 中国語（簡体字）
    - 中国語（繁体字）
4. 保存をクリック

### フォントサイズ調整

読みやすさを調整する最速の方法：

1. メインウィンドウで**フォントサイズ**ボタンをクリック。（ボタンの位置は[メインウィンドウフォントサイズスライダー](/docs/main-window#message-log-font-size-slider)ドキュメントを参照）
2. スライダーウィンドウが表示される
3. スライダーを調整してメッセージログフォントサイズを変更
4. デフォルトサイズは100

:::tip[ヒント]
フォントサイズが大きいほど、メッセージが読みやすくなりますが、表示されるメッセージ数が減ります。
:::

## 外観設定

### 外観設定にアクセス

1. コンフィグウィンドウを開く（ギアアイコンをクリック）
2. **外観**タブに移動
3. 外観オプションを構成

詳細な外観設定については、[外観コンフィグガイド](/docs/config-appearance)を参照してください。

## カスタマイズオプション

### ウィンドウ設定

**ウィンドウ透明度**：
- メインウィンドウの不透明度を調整
- VRCTを半透明にする
- VRChatの背後を見るのに役立つ
- 範囲：0%（非表示）から100%（完全不透明）

### フォントカスタマイズ

**UI/メッセージフォント**：
- フォントファミリーを選択
- UI フォントサイズ 40%-200%
- メッセージログフォントサイズ 40%-200%

### UI要素

**送信メッセージボタンを表示/非表示**：
- 入力フィールドの送信ボタンの表示を切り替え

**再送信ボタンを表示/非表示**：
- ログフィールドの再送信ボタンの表示を切り替え
