# 音声からテキスト（Voice2Chatbox）

音声からテキスト機能（Voice2Chatbox）を使用すると、マイクに話しかけ、VRCTが音声をテキストに変換し、翻訳してVRChat chatboxに送信できます。

## 概要

メッセージをタイプする代わりに、マイクに向かって話すだけです。VRCTは以下を行います：
1. 選択したマイクから音声をキャプチャ
2. 音声をテキストに文字起こし
3. テキストを翻訳（翻訳が有効な場合）
4. メッセージをOSC経由でVRChat chatboxに送信

## 有効にする方法

### Voice2Chatboxを切り替え

1. メインウィンドウで**Voice2Chatbox**トグルスイッチを探す  
![Voice2Chatbox Toggle](./img/voice2chatbox-toggle-off.png)  
2. クリックしてマイク文字起こしを有効にする  
![Voice2Chatbox Toggle](./img/voice2chatbox-toggle-on.png)  
3. トグルはアクティブになるとオンになる

:::warning[重要]
この機能を有効にする前に、VRCT設定で正しいマイクデバイスが選択されていることを確認してください。
:::

## コンフィグレーション

### マイクデバイスを選択

1. コンフィグウィンドウを開く（ギアアイコンをクリック）
2. **デバイス**タブに移動
3. ドロップダウンリストから希望するマイクを選択
4. 保存をクリック

詳細なデバイス構成については、[デバイスコンフィグガイド](/docs/config-device)を参照してください。

### 文字起こし設定を調整

1. コンフィグウィンドウを開く
2. **文字起こし**タブに移動
3. 次のような設定を構成：
   - 文字起こしエンジン
   - AIモデル（該当する場合）
   - 感度
   - ノイズリダクション

詳細な文字起こし設定については、[文字起こしコンフィグガイド](/docs/config-transcription)を参照してください。

## 使い方

### 基本的な使い方

1. Voice2Chatboxトグルを有効にする
2. マイクに話しかける
3. 音声がリアルタイムで入力フィールドにテキストとして表示される
4. 話を終えると、メッセージは自動的に処理される

### 翻訳と共に

1. 翻訳とVoice2Chatboxの両方のトグルを有効にする
2. 「あなたの言語」を話す言語に設定
3. 「対象言語」を希望の言語に設定
4. マイクに話しかける
5. 音声は自動的に文字起こしされて翻訳される

## 機能

### リアルタイム文字起こし

話している間、入力フィールドでテキストがリアルタイムで文字起こしされるのを見ることができます。

### 複数言語サポート

Voice2Chatboxは複数の言語での文字起こしをサポートしています：
- 英語
- 日本語
- 中国語
- 韓国語
- スペイン語
- フランス語
- ドイツ語
- その他多数...

:::tip[ヒント]
文字起こし設定で話す言語を選択すると、精度が向上します。
:::

## ベストプラクティス

### より良い精度のために

1. **はっきり話す**: 言葉をはっきりと、適切な速度で発音する
2. **バックグラウンドノイズを減らす**: 静かな環境を使用するか、ノイズキャンセリングマイクを使用する
3. **適切なマイク位置**: マイクを口から適切な距離に保つ
4. **高品質なマイクを使用**: より良いハードウェアは文字起こし精度を向上させる

### VRChat使用のために

1. **VRChat OSCを構成**: VRChat OSCが有効になっていることを確認
2. **マイクミュートをチェック**: VRCTマイクとVRChatボイスチャットを混同しない
3. **文字数制限を監視**: VRChat chatboxは144文字制限がある
4. **ミュート同期を使用（オプション）**: 必要に応じてミュート同期設定を構成

## トラブルシューティング

### マイクが機能しない

- 設定で正しいマイクが選択されているか確認
- Windowsでマイクがミュートされていないか確認
- Windowsの設定でマイクをテスト

### 文字起こし精度が低い

- 文字起こし設定で正しい言語を選択
- バックグラウンドノイズを減らす
- より明確かつ適切な速度で話す
- 異なる文字起こしエンジンを試す
- マイク品質と位置を確認

### 文字起こしに遅延がある

- システムリソースを確認（CPU/GPU使用率）
- 不要なアプリケーションを閉じる
- より軽い文字起こしモデルを試す
- 安定したインターネット接続を確認（クラウドベースのエンジンの場合）

### メッセージがVRChatに送信されない

- VRChat OSCが有効になっているか確認
- VRCTのOSC設定を確認
- VRChatとVRCTを再起動
- [OSC統合ガイド](./osc-integration)を参照

## プライバシー考慮事項

### ローカル処理対クラウド処理

- **ローカルエンジン**: コンピュータ上で処理が発生、より多くのプライバシー
- **クラウドエンジン**: 音声が外部サーバーに送信される可能性がある

文字起こし設定であなたのプライバシー設定に基づいて適切なエンジンを選択してください。

## パフォーマンスのヒント

### CPU/GPU使用量を減らす

- より軽い文字起こしモデルを使用
- 使用しないときは無効化
- その他のリソース集約的なアプリケーションを閉じる

### 応答時間を改善

- クラウドベースのエンジンを使用（インターネットが必要）
- 十分なシステムリソースを確保
- Windows音声設定を最適化
