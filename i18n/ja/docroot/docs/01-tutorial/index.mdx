# チュートリアルへようこそ

このチュートリアルでは、VRCTの基本的な機能と機能性を案内します。
以下の手順に従って開始してください：

## VRCTとは？
VRCTは、チャットまたは音声翻訳を提供することにより、異なる言語を話す人々の間での会話をサポートするソフトウェアです。これらの機能はVRChat内での使用向けに設計されています。*サポートされていませんが、映画鑑賞などの他の目的にも使用されています。

## 主な機能
- 💬 **チャット機能**: VRChat内でのテキストチャットを可能にします。
- 🌐 **リアルタイム翻訳**: VRChatの音声会話をリアルタイムで翻訳し、テキストとして表示します。
- 🎙/🔈️ **マイク/スピーカー文字起こし**: 音声をテキストに変換し、会話内容を記録します。
- 🌍 **多言語サポート**: 複数の言語をサポートして、異なる言語を話すユーザー間のコミュニケーションを支援します。
- ⚙️ **その他の機能**: ユーザーエクスペリエンスを向上させるためのさまざまな設定とカスタマイズオプションが含まれています。
