# その他タブ
VRCTコンフィグのその他タブでさまざまな雑多な設定をカスタマイズします。

## 一般
![config-other-general.png](./img/config-other-general.png)

- **送信後にメッセージボックスを自動クリア**: 送信後のメッセージボックスの自動クリアを有効または無効にします。
- **翻訳されたメッセージのみを送信**: 元のテキストと翻訳の両方を送信するか、翻訳されたメッセージのみを送信するかを選択します。
- **メッセージログを自動保存**: メッセージログの自動保存を有効または無効にします。
    - ![File Icon](./img/config-other-general-file-icon.png)：メッセージログが保存されるディレクトリを開きます。
- **VRC マイクミュート同期**: VRChatとVRCT マイク文字起こし間のミュートステータスの同期を有効または無効にします。
- **メッセージをVRChatに送信**: VRChatへのメッセージ送信を有効または無効にします。

## サウンド
![config-other-sounds.png](./img/config-other-sounds.png)
- **チャットメッセージ送信時に通知音を有効にする**: VRChatでチャットメッセージを送信するときの通知音を有効または無効にします。

## Speaker2Chatbox
![config-other-speaker2chatbox.png](./img/config-other-speaker2chatbox.png)
- **受信したメッセージをVRChatに送信**: 受信したメッセージをVRChat chatboxに送信するのを有効または無効にします。

## メッセージフォーマット
VRChatに送信されるときのメッセージフォーマット方法を定義します。

### メッセージ送信フォーマット
VRChatに送信するときのチャットまたはマイク文字起こしメッセージフォーマット。
![config-other-message-format-send.png](./img/config-other-message-format-send.png)

### Speaker2Chatboxメッセージフォーマット
[Speaker2Chatbox](./config-other#speaker2chatbox)を有効にした場合、VRChatに送信するときの受信したメッセージのフォーマット方法を定義します。
![config-other-message-format-speaker2chatbox.png](./img/config-other-message-format-speaker2chatbox.png)

### 音訳
![config-other-other.png](./img/config-other-transliteration.png)
- **ローマ字を表示**: 日本語テキストのローマ字表示を有効または無効にします。
- **ひらがなを表示**: 日本語テキストのひらがな表示を有効または無効にします。
