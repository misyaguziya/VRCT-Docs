# コンフィグウィンドウの概要

コンフィグウィンドウを使用すると、外観から翻訳エンジンまで、VRCTのさまざまな側面をカスタマイズできます。   
メインウィンドウの**ギアアイコン（⚙️）**をクリックして開きます。

![config-window-overview.png](./img/config-window-overview.png)

## デバイスタブ
オーディオ入力および出力デバイスを管理します。  
詳細については、[デバイスタブ](./config-device)ページを参照してください。

## 外観タブ
アプリケーションのルック・アンド・フィールをカスタマイズします。  
詳細については、[外観タブ](./config-appearance)ページを参照してください。

## 翻訳タブ
翻訳エンジンを構成します。  
詳細については、[翻訳タブ](./config-translation)ページを参照してください。

## 文字起こしタブ
文字起こしエンジンを構成します。  
詳細については、[文字起こしタブ](./config-transcription)ページを参照してください。

## VRタブ
VR関連（主にオーバーレイについて）の設定を構成します。  
詳細については、[VRタブ](./config-vr)ページを参照してください。

## その他のタブ
その他の雑多な設定を構成します。  
詳細については、[その他タブ](./config-other)ページを参照してください。

## ホットキータブ
VRCTのさまざまなアクションのキーボードショートカットを構成します。  
詳細については、[ホットキータブ](./config-hotkeys)ページを参照してください。

## プラグインタブ
VRCTプラグインを管理するための構成オプション。  
詳細については、[プラグインタブ](./config-plugins)ページを参照してください。

## 詳細設定タブ
VRCTの詳細構成オプション。  
詳細については、[詳細設定タブ](./config-advanced)ページを参照してください。
