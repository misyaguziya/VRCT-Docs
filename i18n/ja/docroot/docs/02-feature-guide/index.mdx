# 機能ガイドへようこそ

この機能ガイドでは、VRCTのさまざまな機能と機能性を説明します。
以下のセクションに従って、詳細情報を確認してください：

## コア機能

### 翻訳機能
- **[リアルタイム翻訳](/docs/feature-guide/translation)** - さまざまな翻訳エンジンを使用してメッセージをリアルタイムで翻訳します
- **[複数の対象言語](/docs/feature-guide/multi-target-language)** - 複数の言語への同時翻訳をサポートしています
- **[翻訳エンジンの選択](/docs/feature-guide/translation-engines)** - さまざまな翻訳エンジンから選択します（Google、DeepL、AIモデルなど）

### 文字起こし機能
- **[音声からテキストへ（Voice2Chatbox）](/docs/feature-guide/voice-to-text)** - マイクを使用して音声をテキストに変換します
- **[スピーカーからテキストへ（Speaker2Log）](/docs/feature-guide/speaker-to-text)** - システムオーディオをテキストに文字起こしします

### コミュニケーション機能
- **[メッセージ送信](/docs/feature-guide/send-message)** - テキストまたは音声で翻訳付きメッセージを送信します
- **[メッセージ受信](/docs/feature-guide/receive-message)** - 他者からのメッセージを受信して翻訳します
- **[OSC統合](/docs/feature-guide/osc-integration)** - チャットボックス機能用のVRChat OSCとの統合

### VR機能
- **[オーバーレイモード](/docs/feature-guide/overlay-mode)** - VR内にVRCTインターフェイスをオーバーレイとして表示します
- **[VRCミュート同期](/docs/feature-guide/vrc-mute-sync)** - VRChatのミュート状態と同期します

### カスタマイズ機能
- **[言語プリセット](/docs/feature-guide/language-presets)** - プリセット言語の組み合わせを素早く切り替えます
- **[外観のカスタマイズ](/docs/feature-guide/appearance)** - VRCTの外観をカスタマイズします
