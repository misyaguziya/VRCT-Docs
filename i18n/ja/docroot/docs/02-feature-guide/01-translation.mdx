# リアルタイム翻訳

VRCTは、言語の壁を超えてコミュニケーションを取るためのリアルタイム翻訳機能を提供します。

## 概要

翻訳機能は、送信メッセージと受信メッセージを異なる言語間で自動的に翻訳します。

## 使い方

### 翻訳を有効化

1. メインウィンドウの**翻訳**トグルスイッチを探す  
![Translation Toggle Off](./img/translation-toggle-off.png)  
2. クリックして翻訳機能を有効にする  
![Translation Toggle On](./img/translation-toggle-on.png)  
3. 有効にすると、送受信されるすべてのメッセージが翻訳されます

:::tip[ヒント]
言語設定を失うことなく、翻訳のオン/オフを素早く切り替えることができます。
:::

### 言語を選択

![Language Selection](./img/language-select.png)

1. **あなたの言語**: 話すまたはタイプする言語を設定
2. **対象言語**: 翻訳したい言語を設定

言語領域をクリックして言語選択メニューを開き、希望の言語を選択します。
![Language Menu](./img/language-menu.png)


### 言語を交換

言語設定の間にあるスワップボタンをクリックして、あなたの言語と対象言語をすばやく切り替えます。

![Language Swap](./img/language-swap.png)

## 翻訳フロー

### メッセージ送信
すでに[チュートリアル メッセージ送信](../send-message)ガイドで説明していますが、簡単な概要を以下に示します：

1. あなたの言語でメッセージをタイプ
2. VRCTは自動的にそれを対象言語に翻訳
3. 元のテキストと翻訳されたテキストの両方がチャット履歴に表示される
4. 翻訳されたメッセージがOSC経由でVRChatchatboxに送信される

### メッセージ受信
すでに[チュートリアル メッセージ受信](../receive-message)ガイドで説明していますが、簡単な概要を以下に示します：


1. 他のプレイヤーが話すとき（スピーカー文字起こしで キャプチャ）
2. VRCTはオーディオをテキストに文字起こし
3. 文字起こしされたテキストが自動的にあなたの言語に翻訳される
4. 元のテキストと翻訳されたテキストの両方がチャット履歴に表示される

## サポートされている言語

VRCTは、次のような幅広い言語をサポートしています：

- 英語
- 日本語
- 中国語（簡体字/繁体字）
- 韓国語
- スペイン語
- フランス語
- ドイツ語
- イタリア語
- ポルトガル語
- ロシア語
- アラビア語
- タイ語
- ベトナム語
- その他多数...

:::info[情報]
利用可能な言語は、選択した翻訳エンジンによって異なる場合があります。  
詳細は[翻訳エンジン選択](./translation-engines)ガイドを参照してください。
:::

## 機能

### 複数言語翻訳
複数の対象言語にメッセージを同時に翻訳します。詳細は[複数の対象言語](./multi-target-language)ガイドを参照してください。

### 翻訳履歴

すべての翻訳はチャット履歴に保存されるため、以前の会話を確認できます。

### カスタマイズ可能な翻訳品質

異なる翻訳エンジンは異なる品質と機能を提供しています。[翻訳エンジン選択](./translation-engines)ガイドであなたのニーズに最適なエンジンを選択できます。

## より良い翻訳のためのヒント
- 文字起こしの精度を向上
    1. **はっきり話す**: 音声入力の場合、はっきり話すと文字起こしの精度が向上
    2. **周囲のノイズをチェック**: スピーカー入力の場合、バックグラウンドノイズを最小化して文字起こし精度を向上
- 翻訳品質を向上
    1. **シンプルな文を使用**: より短く、シンプルな文は通常、より良く翻訳される
    2. **適切なエンジンを選択**: 異なるエンジンは異なる言語ペアでより良いパフォーマンスを発揮

## トラブルシューティング

### 翻訳が機能しない

- 翻訳トグルが有効になっているか確認
- 両方の言語が正しく選択されているか確認

### 翻訳品質が低い

- 別の翻訳エンジンに切り替えしてみる
- メッセージを簡潔にする
