# 複数の対象言語

VRCTを使用すると、メッセージを最大3つの異なる対象言語に同時に翻訳できます。これは多言語グループの会話に最適です。

## 概要

異なる言語を話す人とVRChat世界で同じコミュニケーションを取る必要がある場合、複数の対象言語機能を使用すると、複数の言語で翻訳を同時に送信できます。

## 使い方

### 対象言語を追加

1. メインウィンドウで言語設定セクションを探す
2. 対象言語の横にある**[+]**ボタンをクリック  
![Add Target Language](./img/add-target-language.png)  
3. メニューから追加の言語を選択  
![Add Target Language](./img/added-target-language.png)  
4. 最大3つの対象言語まで繰り返し追加

### 対象言語を削除

1. 削除したい対象言語の横にある**[-]**ボタンをクリック
![Remove Target Language](./img/remove-target-language.png)
2. 言語は直ちに削除される

## メッセージ表示

複数の対象言語が設定されている場合：

1. 元のメッセージが最初に表示される
2. 翻訳は設定した対象言語の順序で表示される

**例**：  
**日本語**から（**英語**、**韓国語**、**スペイン語**）に設定されている場合  
![Example Send Message Display Multi Languages](./img/example-send-message-display-multi-languages.png)

:::tip[ヒント]
翻訳の順序は、上から下へ設定した対象言語の順序と一致します。
:::

## ユースケース

### 多言語VRChat世界

異なる国のプレイヤーがいる国際的な世界を訪問する場合：
- その世界で使用されるすべての一般的な言語を設定
- すべてのユーザーが彼らの好みの言語で翻訳を見ることができる

### イベント主催

国際的な参加者がいるイベントを主催する場合：
- あなたの期待される視聴者の主要言語をすべて追加
- 重要なお知らせが皆に届く

### 言語学習

複数の言語を練習する場合：
- メッセージがさまざまな言語にどのように翻訳されるかを見る
- 異なる対象言語間で翻訳を比較する

## パフォーマンス考慮事項

### 翻訳速度

- より多くの対象言語= より多くのAPI呼び出しまたは処理
- クラウドベースのエンジン：複数のAPIリクエスト
- AIエンジン：シーケンシャル処理

:::info[情報]
3つの対象言語を使用すると、1つの対象言語を使用するより約3倍長くなります。
:::

### API制限

クラウドベースの翻訳エンジンを使用している場合：
- 各対象言語は個別のAPI呼び出しとしてカウント
- APIクォータ制限がある場合はAPI使用状況を監視
- 無料のAIエンジンと有料APIを混合することを検討

## ベストプラクティス

1. **本質的な言語から始める**: 実際に必要な言語のみを追加
2. **パフォーマンスを監視**: 翻訳が遅すぎる場合は対象言語を減らす
3. **適切なエンジンを選択**: 各言語ペアに最適なエンジンを使用
4. **必要に応じて更新**: 世界に誰がいるかに基づいて言語を追加/削除

## 制限事項

- 最大3つの対象言語を同時に設定可能
- 各言語は元の言語と異なる必要がある

## トラブルシューティング

### 翻訳に時間がかかりすぎている

- 対象言語の数を減らす
- より高速な翻訳エンジンに切り替える（Google、Bing）
- ネットワーク接続を確認（クラウドエンジンの場合）

### 対象言語が表示されない
- 翻訳エンジンがその言語ペアをサポートしているか確認

### メッセージがVRChat Chatboxに長すぎる

- VRChat chatboxは文字数制限があります（144文字）
- 複数の翻訳がこの制限を超える可能性があります
- 文字制限を超えた場合、メッセージが正しく表示されない可能性があります。必要に応じて対象言語の数を減らすか、メッセージを短くしてください。
